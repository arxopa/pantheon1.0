# Pantheon 1.0 Release Announcement

## Short Version

Pantheon 1.0 is now public.

The repository is live at `https://github.com/arxopa/pantheon1.0`, the `main` branch is protected, and release `v1.0.0` is published.
This baseline includes the clustered runtime, the operator control plane, Atman personalities, Pantheon Ultra expert routing, and guarded self-learning.

## Standard Version

Pantheon 1.0 is the first public baseline of the Pantheon runtime.
It brings together the React control plane, the Node.js agent runtime, Atman personality management, Ultra multi-expert orchestration, guarded self-learning, validation surfaces, and operator-facing admin tools in one public repository.

What is included in this release:

- Pantheon Ultra with expert routing, synthesis, contradiction handling, and safety checks;
- operator-visible admin smoke flow for `!ultra` and `!normal` plus recent Ultra session inspection;
- Atman dialogue, personality mutation, event auditing, social simulation, and multimodal stubs;
- learning, validation, navigation, research, integration, and training surfaces wired into a single runtime;
- public release notes, repository metadata, and branch protection for the published baseline.

Validated before release with:

- `npm run beta:admin`
- `npm run beta:test`
- `npm run beta:scenarios`
- `npm run check`
