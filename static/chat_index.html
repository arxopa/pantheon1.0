Pantheon / Atman

Чат с Атманом

Этот интерфейс ведет прямой диалог с ядром Atman. Полный runtime Pantheon по-прежнему доступен через `/api/agent/run`, но здесь фокус на ребенке и его собственной памяти.

Ожидание первого запроса.
Стартовые реплики для семейного сценария:
Личность Атмана определяет отдельную память, веса и собственную траекторию самообучения.
`atman-direct` работает с памятью и весами Атмана. `pantheon-runtime` проходит через полный стек Pantheon, включая validator и внешние инструменты.
В режиме `streaming-on` Atman печатает ответ постепенно через SSE endpoint `/api/atman/chat/stream`.
